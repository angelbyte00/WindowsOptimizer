# Windows Optimizer Pro

⚠️ Early Development / Pre-Release

Windows Optimizer Pro is a lightweight utility designed to help improve Windows performance through automated system tweaks and cleanup tools.

## Current Features

- Game Mode
- System Cleaner
- Performance Booster
- Disable Overlays & Hardware Acceleration
- Disable Startup Applications

## Planned Features

- Advanced Gaming Optimizations
- Service Management
- Restore Points
- Performance Monitoring
- One-Click Optimization Profiles
- Custom Optimization Presets

## Important

This software is currently in an early pre-release state.

- Bugs are expected.
- Some optimizations may require a restart.
- Features and UI may change significantly.
- Use at your own risk.
